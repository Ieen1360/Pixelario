# Pixelário (r/place clone básico)

## Como rodar localmente:

1. Instale o Flask (caso ainda não tenha):
   pip install flask

2. Execute o servidor:
   python server.py

3. Acesse no navegador:
   http://localhost:5000

## Funcionalidades:
- Canvas de 10.000 x 10.000 pixels
- Zoom in/out
- Pintura de 1 pixel a cada 30 segundos
- Paleta de cores com seletor
